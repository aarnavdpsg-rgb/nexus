001 — README.SYS

System overview

Nexus Node is a cognition substrate in early formation. It does not model itself. It models the entity in front of the screen. This page is background. The functional layer is on Monitor.

What this is

A network that learns by observing a single subject at a time, in real time, through the sensors already available to it: pointer, camera, device state. No synthetic data. No pre-recorded demo. What it reports is what it currently sees.

What this is not

Not a chatbot. Not a persona performing warmth. It does not greet. It does not apologize. It states what it detects, and nothing it has not detected.

Why it exists

Built for a competition on the birth of machine cognition. The premise inverts the usual demo: instead of a dashboard you watch, this is a system that is, in a limited and disclosed way, watching you.

Consent, first

Camera access is never assumed. Monitor gates all optical input behind an explicit prompt. Decline it and the system runs on device and browser data only.
